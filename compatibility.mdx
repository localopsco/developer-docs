## Cloud providers

LocalOps supports **Amazon Web Services** at the moment.

Coming soon:

- Microsoft Azure
- Google Cloud Platform

## Regions

LocalOps supports all AWS cloud regions where EKS is supported.

As on June 1, 2025, AWS has 33 global regions. Learn more in
[AWS site](https://aws.amazon.com/about-aws/global-infrastructure/).
